# Guía de Contribución

¡Gracias por tu interés en colaborar con **Summan AI - Journey to the Digital Twin**! Queremos que contribuir sea lo más sencillo y transparente posible.

## Cómo empezar

1. **Fork** el repositorio.
2. Crea una **Rama** para tu funcionalidad o corrección (`git checkout -b feature/nueva-funcionalidad`).
3. Realiza tus **Commits** siguiendo los estándares convencionales.
4. Asegúrate de que el código sigue el estilo del proyecto.
5. Envía un **Pull Request**.

## Reportar Bugs

Si encuentras un error, por favor utiliza la plantilla de **Bug Report** en GitHub. Incluye:
- Pasos para reproducir.
- Comportamiento esperado y obtenido.
- Información sobre tu entorno.

## Feature Requests

Estamos abiertos a nuevas ideas. Utiliza la plantilla de **Feature Request** para proponer cambios importantes.

## Estándares de Código

- Usamos Javascript/Node.js para los scripts centrales.
- Las Skills deben seguir el formato de Markdown definido en `skills/`.
- Mantén la documentación actualizada en la carpeta `docs/`.

---

Al contribuir, aceptas que tu código estará bajo la licencia **MIT**.
