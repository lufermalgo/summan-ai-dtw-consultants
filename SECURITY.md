# Política de Seguridad

## Versiones Soportadas

Actualmente, solo la última versión de la rama principal (`main`) recibe actualizaciones de seguridad.

| Versión | Soportada |
| ------- | --------- |
| main    | ✅        |
| < 1.0.0 | ❌        |

## Reportar una Vulnerabilidad

Valoramos mucho la seguridad de este proyecto. Si descubres una vulnerabilidad de seguridad, por favor no abras un issue público. En su lugar, comunícate directamente con los mantenedores del repositorio.

Estamos comprometidos a:

1. Reconocer la recepción de tu reporte en menos de 48 horas.
2. Trabajar contigo para validar y corregir el problema.
3. Darte crédito por el hallazgo una vez que se haga público el fix.

Gracias por ayudarnos a mantener este proyecto seguro.
